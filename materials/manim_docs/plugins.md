# Plugins Reference

Plugins extend Manim's functionality for specific use cases.

## Popular Plugins
- **`manim-physics`**: Adds 2D and 3D physics engines (Rigid bodies, Electromagnetism, etc.).
- **`manim-voiceover`**: Synchronizes animations with generated or recorded voiceovers (very useful for 3B1B style!).
- **`manim-chemistry`**: Tools for rendering chemical structures and reactions.
- **`manim-rubikscube`**: Specifically for Rubik's Cube animations.

## How to Use
1. Install via pip: `pip install manim-voiceover`
2. Import in your script: `from manim_voiceover import VoiceoverScene`
3. If necessary, enable in `manim.cfg`.

## Discovery
Search for `manim-` on [PyPI](https://pypi.org/) to find more community plugins.
