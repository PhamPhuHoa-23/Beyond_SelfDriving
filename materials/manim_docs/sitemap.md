# Manim Module Sitemap (Technical Reference)

This sitemap lists the major modules and sub-modules in the Manim Community library. You can use this as a map for what is available.

## Core Modules
- `manim.animation`: Base classes and standard animations (Creation, Fading, Growing, etc.).
- `manim.mobject`: Base classes for all objects on screen.
- `manim.scene`: The canvas and animation control.
- `manim.camera`: Camera settings and movement.

## Mobject Sub-modules
- `manim.mobject.geometry`: Basic shapes (Circles, Squares, Lines, etc.).
- `manim.mobject.graphing`: Axes, function graphs, coordinate systems.
- `manim.mobject.text`: Text and LaTeX (MathTex).
- `manim.mobject.table`: Data tables and MathTables.
- `manim.mobject.matrix`: Linear algebra matrices.
- `manim.mobject.three_d`: 3D objects and surfaces.
- `manim.mobject.svg`: Loading SVG icons and drawings.
- `manim.mobject.types`: specialized types (ImageMobject, PMobject).
- `manim.mobject.probability`: Probability visualizations (SampleSpace, BarChart).

## Specialized Tools
- `manim.utils`: Colors, space operations, and helper functions.
- `manim.renderer`: Rendering engines (Cairo, OpenGL).
- `manim.plugins`: External extensions.

---
*Note: This sitemap covers the primary modules used in 3Blue1Brown-style animations. For the absolute full list of every internal function, refer to the [Online API Reference](https://docs.manim.community/en/stable/reference.html).*
